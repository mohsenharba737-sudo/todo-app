📌 *المميزات*


إضافة مهمة جديدة

حذف أي مهمة

تعديل المهمة مباشرة

وضع الليلي (Dark Mode)

حفظ المهام بعد إغلاق الصفحة

دعم زر Enter لإضافة المهمة





⚡ *طريقة الاستخدام*


افتح ملف index.html في أي متصفح
.
اكتب المهمة في الحقل واضغط إضافة أو زر Enter.

اضغط على المهمة لوضع علامة "مكتملة".

استخدم زر تعديل لتغيير نص المهمة.

استخدم زر حذف لمسح المهمة.

استخدم زر الوضع الليلي 🌙 لتغيير ثيم الصفحة.
