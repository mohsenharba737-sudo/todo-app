📌 Features



Add a new task

Delete any task

Edit a task directly

Dark Mode

Save tasks after closing the page

Support the Enter key to add a task






⚡ How to use



Open the index.html file in any browser.

Type your task in the input field and press Add or the Enter key.

Click on a task to mark it as "Completed".

Use the Edit button to change the task text.

Use the Delete button to remove a task.

Use the 🌙 Dark Mode button to change the page theme.
