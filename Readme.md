# Todo App

مشروع **Todo App** لتعلم HTML, CSS، وJavaScript.

---

## 📌 المميزات
- إضافة مهمة جديدة  
- حذف أي مهمة  
- تعديل المهمة مباشرة  
- وضع الليلي (Dark Mode)  
- حفظ المهام بعد إغلاق الصفحة  
- دعم زر Enter لإضافة المهمة  

---

## ⚡ طريقة الاستخدام
1. افتح ملف `index.html` في أي متصفح.  
2. اكتب المهمة في الحقل واضغط **إضافة** أو زر **Enter**.  
3. اضغط على المهمة لوضع علامة "مكتملة".  
4. استخدم زر **تعديل** لتغيير نص المهمة.  
5. استخدم زر **حذف** لمسح المهمة.  
6. استخدم زر **الوضع الليلي 🌙** لتغيير ثيم الصفحة.  

---

## 💻 التقنيات المستخدمة
- HTML  
- CSS (تصميم واستجابة)  
- JavaScript (DOM + LocalStorage + التفاعل مع المستخدم)  

---

## 📂 ملاحظات
- كل المهام يتم حفظها تلقائيًا في المتصفح (LocalStorage).  
- يمكنك نسخ المشروع أو تطويره بسهولة.
